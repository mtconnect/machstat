Steps:

1. Open the main.exe standalone under administrative cmd.

2. Wait untill the server is started on localhost and then hit Enter.

3. Enter valid mtconnect agent link and hit enter, repeat this as many times as needed. E.g. http://mtconnect.mazakcorp.com:5609/current

4. In google chrome browser open up "localhost:5000"

5. To go to the list of machines that have been connected after step 3, hit the play button and then the browser directs you to 'localhost:5000/machstat_machines'.

6. Navigate to the agent or the dashboard for any machine on the list by clicking on the respective links.
